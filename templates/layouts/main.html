Match Algorithm

We need to define what good matching is

We went through all event types in the activities dataset, and decide MessageSent could be a good criterion.If the user sent message to another use, we assume they are matched successfully. More messages sent between two users mean that the these users are more matched.

How to find similar users? Clustering.

We combined user profile and user activity dataset to get features for each user. Then with these features, we run the kmeans clustering algorithm to put similar users together.
